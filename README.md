<h1>Task Manager</h1>

<h2>📃Resumo</h2>

<p>Aplicação web para gerenciamento de tarefas de forma simples e prática</p>

<h2>⚙️Funções</h2>

<ul>
    <li>Login e cadastro de usuário</li>
    <li>Cadastrar tarefa</li>
    <li>Editar tarefa</li>
    <li>Remover tarefa</li>
</ul>

<h2>💻Tecnologias</h2>

<ul>
    <li>Laravel 11</li>
    <li>PHP 8.2</li>
    <li>Bootstrap 5.3</li>
    <li>MySQL</li>
</ul>

<p>Códigos para rodar:</p>

...

...

```php artisan serve```
